
# 🎲 Number Randomizer - JavaScript

Projeto desenvolvido com **HTML, CSS e JavaScript**, com o objetivo de realizar o sorteio de números aleatórios dentro de um intervalo definido pelo usuário.

Este projeto foi criado para praticar **lógica de programação**, **manipulação do DOM** e **interações dinâmicas com JavaScript**.

---

## 🚀 Funcionalidades

- Definição de valor mínimo e máximo
- Geração de número aleatório entre os valores informados
- Interface simples e intuitiva
- Feedback imediato ao usuário

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
